# Security Policy

## Supported Versions

Market Loom is in early development. Security fixes target the current `main` branch until the project publishes versioned releases.

## Reporting a Vulnerability

Please report vulnerabilities through the GitHub repository security reporting flow when available, or open a minimal issue that avoids disclosing secrets or exploit details.

Do not include provider tokens, account identifiers, private server addresses, generated databases, or real market data in reports.
